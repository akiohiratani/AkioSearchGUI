🏇 AkioScraper(レース結果収集アプリ) 取扱説明書

生成AI時代に必須のデータ収集ツール

⚡ ChatGPT・Claude等の生成AIで競馬予想の精度が劇的に向上

データなしのAI予想：一般的な競馬知識のみで的中率は10-15%程度

豊富なデータ付きAI予想：的中率が35-40%以上に向上！

本アプリで収集したCSVファイルをアップロードするだけで、AIが高精度な予想を提供します。

アプリの特徴

🎯 G1・重賞レース特化

今週のメインレース（G1・重賞）に特化し、1クリックで過去分のデータを収集

🤖 生成AI最適化

ChatGPT・Claudeがそのまま読み込める形式でデータを出力

📊 データ分析対応

Excel、Python、機械学習ライブラリで即座に分析可能

⚡ 超簡単操作

レースを選んで期間を選ぶだけの2ステップ操作

インストールと起動

  1. ダウンロードしたファイルを解凍してください。
  2. Start.exeファイルをダブルクリックして起動してください。
    ※Google Chromeがインストールされていることを推奨します。
    ※セキュリティ警告が表示された場合は「実行」または「許可」をクリックしてください。

データ収集手順（たった3ステップ！）

  1. アプリ起動後、「レースから選択」ボタンをクリック
  2. 今週のG1・重賞レース一覧から目的のレースを選択
  3. 取得年数（1年〜15年）を選んでOKをクリック

※収集には数分かかります。処理完了後、CSVファイルの保存場所が表示されます。

出力されるCSVファイルについて

ファイル構成

  • [レース名]_分析データ.csv：過去の開催データと今週の出走予定馬の基本情報が含まれます
  • [レース名]_出走する馬の戦歴データ.csv：今週出走する各馬の過去レース成績が含まれます

生成AI向け最適化

  • 全てのカラム名が日本語で、AIが理解しやすい形式
  • 文字コードはUTF-8で日本語の文字化けを防止
  • 「データ種別」列で過去データと今週の出走馬を区別
  • 日付形式や特殊記号も生成AIが適切に解釈できる形式

生成AIで競馬予想を革新する使い方

🤖 ChatGPT・Claude活用法

基本的な使い方：

  1. 収集した2つのCSVファイルをChatGPTやClaudeにアップロード
  2. 以下のようなプロンプトを入力
2つのCSVファイルをアップロードしました： 1. 「[レース名]_分析データ.csv」には過去の開催データと今週の出走予定馬の情報が含まれています。「データ種別」列が「過去」のデータは過去の結果、「今週」は出走予定馬です。 2. 「[レース名]_出走する馬の戦歴データ.csv」には今週出走する馬の過去レース成績が含まれています。 これらのデータを分析して、以下を教えてください： 1. 過去のレースで好成績を収めた馬の共通点 2. 血統と成績の相関 3. 騎手と馬場の相性 4. 今週の有力馬と穴馬候補 最後に、軸馬1頭と相手3〜4頭、穴馬1〜2頭を挙げて、推奨買い目も提案してください。

データがあることで可能になること：

  • 過去の波乱レースのパターン認識
  • 特定の馬場状態での好走馬の特徴分析
  • 人気馬が崩れやすい条件の特定
  • 血統×馬場×展開の複合分析
  • 各馬の適性と当日条件のマッチング

📊 Excel・Python活用法

Excelでの基本分析例：

  • 人気順別の勝率・連対率・複勝率の自動計算
  • 血統別の成績集計（サンデーサイレンス系、ディープインパクト産駒など）
  • 枠順による有利不利の可視化
  • オッズ帯別の回収率分析

Pythonでの高度な分析例：

import pandas as pd import matplotlib.pyplot as plt # 分析データと戦歴データを読み込み df_race = pd.read_csv('[レース名]_分析データ.csv') df_history = pd.read_csv('[レース名]_出走する馬の戦歴データ.csv') # 過去データのみ抽出 past_data = df_race[df_race['データ種別'] == '過去'] # 人気順別の的中率を計算 win_rate = past_data.groupby('人気')['着順'].apply( lambda x: (x.astype(str) == '1').mean() ) # グラフ化 plt.figure(figsize=(10, 6)) plt.bar(win_rate.index, win_rate.values) plt.title('人気順別勝率') plt.xlabel('人気') plt.ylabel('勝率') plt.show()

機械学習モデル構築例：

  • ランダムフォレストによる着順予測
  • ロジスティック回帰による勝ち馬予測

よくある質問

Q: どのレースのデータを収集できますか？

A: 今週開催されるG1・重賞レースのデータを収集できます。地方競馬や平場レースには対応していません。

Q: データ収集にはどのくらい時間がかかりますか？

A: レースや年数により異なりますが、10年分のデータで約3〜5分程度です。15年分でも10分以内に収集できます。

Q: 生成AIにファイルをアップロードする際の注意点はありますか？

A: 2つのファイルを同時にアップロードし、プロンプトでは各ファイルの内容を明確に説明すると、AIがより正確に分析できます。
